Unified Internship & Mentorship Portal (UIMP)
1. Introduction

The Unified Internship & Mentorship Portal (UIMP) is a platform designed to help students find internships, connect with mentors, and track their learning progress.
This user testing was conducted to evaluate the usability, functionality, and overall user experience of the portal.

2. Objectives of User Testing

To check whether users can easily navigate the portal

To verify core features such as registration, login, and browsing internships

To identify usability issues and bugs

To understand user satisfaction and areas for improvement

3. Test Environment

Device: Laptop / Desktop

Operating System: Windows

Browser: Google Chrome

Internet Connection: Stable broadband connection

4. User Profile

User Type: Student

Experience Level: Beginner to Intermediate

Goal: Find internships and connect with mentors

5. Test Scenarios & Results
Scenario 1: User Registration

Steps:

Open the UIMP homepage

Click on the “Sign Up” button

Enter valid user details

Submit the registration form

Expected Result:
User should be registered successfully and redirected to the dashboard.

Actual Result:
Registration completed successfully.

Status: ✅ Pass

Scenario 2: User Login

Steps:

Open login page

Enter registered email and password

Click “Login”

Expected Result:
User should be logged in and see the dashboard.

Actual Result:
Login works as expected.

Status: ✅ Pass

Scenario 3: Browse Internships

Steps:

Navigate to Internship section

Apply filters (domain, duration)

View internship details

Expected Result:
Internships should be displayed clearly with details.

Actual Result:
Internships load correctly, filters work properly.

Status: ✅ Pass

Scenario 4: Mentor Interaction

Steps:

Open Mentorship section

View mentor profiles

Send a mentorship request

Expected Result:
Request should be sent successfully.

Actual Result:
Request sent, confirmation message displayed.

Status: ✅ Pass

6. Issues Identified

Page loading is slightly slow on the internship listing page

No tooltip/help text for first-time users

7. User Feedback

The interface is clean and easy to understand

Navigation is simple

Dashboard gives a clear overview

Search and filters are useful

8. Recommendations

Improve page loading speed

Add onboarding tips for new users

Provide notifications for mentor responses

9. Conclusion

The Unified Internship & Mentorship Portal (UIMP) is user-friendly and meets its core objectives. Most features work as expected, and the overall experience is positive. With minor improvements, the platform can offer an even better experience to users.
